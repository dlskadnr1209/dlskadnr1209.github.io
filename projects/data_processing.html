Data Processing System

Introduction

 With Korea participating in the next-generation large radio telescope project, there was a need to build a system to store massive amounts of data and make it accessible for Korean researchers. I had the opportunity to assist in the initial stages of this system's construction, where I learned a lot about Docker and containers through the Korea Institute of Information Technology. As part of the project, I created Docker containers for the software used by observational researchers and tested deployment. In addition, I worked hands-on with Kubernetes and OpenStack, learning how to manage and deploy large-scale infrastructure.
